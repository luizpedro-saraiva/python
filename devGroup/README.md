# _devGroup_

**Aprender juntos, codar juntos, evoluir juntos.**

## Sobre

O **devGroup** é um grupo de estudos voltado para desenvolvimento de software, criado para reunir pessoas que desejam evoluir tecnicamente com constância, prática e colaboração.

Este repositório foi criado para armazenar códigos, exercícios, desafios e projetos desenvolvidos ao longo da jornada de estudos do grupo.

O objetivo deles é transformar teoria em prática, desenvolver lógica de programação, fortalecer fundamentos técnicos e construir experiência real por meio do desenvolvimento colaborativo.
[https://devgroup.com.br]
